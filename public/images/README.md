# Logo Files

Place your Terra Trionfo logo files here:

- **logo.png** or **logo.svg** - Main logo (for light backgrounds)
- **logo-white.png** or **logo-white.svg** - White version (for dark backgrounds) [optional]
- **favicon.ico** - Browser favicon [optional]

## Recommended Formats
- SVG (scalable, best quality)
- PNG with transparent background
- Dimensions: 200x200px minimum

Once uploaded, the logo will automatically appear in:
- Header navigation
- Footer
- Landing page hero
- Authentication pages
